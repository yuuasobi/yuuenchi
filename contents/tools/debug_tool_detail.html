🐛 Debug Tool

📖 使い方

Debug Toolは、Windowsデスクトップアプリケーションの自動操作とクラッシュ監視を行うPythonツールです。GUI版とバックグラウンド実行版の両方を提供します。

GUI版の基本操作

  1. アプリケーション設定（アプリパス、ウィンドウタイトル）を入力
  2. 「アプリ起動・監視開始」ボタンをクリック
  3. 「自動操作開始」ボタンでGUI自動操作を実行
  4. 「バックグラウンド実行」ボタンでバックグラウンド処理を実行
  5. 「監視停止」ボタンでアプリを終了・監視を停止

バックグラウンド版の使用方法

  1. 設定ファイル`debug_config.json`を編集
  2. バッチファイルまたはPowerShellスクリプトで実行
  3. `debug_log.txt`でログを確認

⚙️ 機能

  • 外部アプリ起動 - 指定したアプリケーションを自動起動
  • GUI自動操作 - メニュー選択やボタンクリックを自動実行
  • リソース監視 - CPU使用率とメモリ使用量を1秒間隔で監視
  • クラッシュ検出 - プロセスの異常終了を検出して通知
  • ログ記録 - 操作履歴とエラーをログに記録
  • バックグラウンド実行 - GUIなしでバックグラウンドでデバッグ処理を実行

🎛️ パラメータ

パラメータ名 入力形式 必須/任意 説明
アプリパス ファイルパス 必須 対象アプリの実行ファイルパス（例: notepad.exe）
ウィンドウタイトル テキスト 必須 アプリのウィンドウタイトル（例: Untitled - Notepad）
監視間隔 数値（秒） 任意 リソース監視の間隔（デフォルト: 1.0秒）
自動操作シーケンス 設定ファイル 任意 実行する自動操作のシーケンスを定義
