🖼️ Image Resize Tool

📖 使い方

Image Resize Toolは、選択したフォルダ（サブフォルダも含む）内のすべての画像ファイルを指定したパーセンテージで一括リサイズし、元のディレクトリ構造を保ったまま新しいフォルダに保存します。

基本的な使用方法

  1. Pythonで`img_resize.py`を実行（またはexe版を利用）
  2. 「Browse」ボタンで画像フォルダを選択
  3. スライダーや入力欄でリサイズ率を設定（10%～200%）
  4. 「Start resizing」ボタンで処理開始
  5. 進捗はプログレスバーで確認
  6. 完了後、出力先フォルダのパスが表示されます

⚙️ 機能

  • 一括リサイズ - サブフォルダも含めた画像の一括リサイズ
  • 多形式対応 - jpg, jpeg, png, bmp, gif, tiff, webpに対応
  • リサイズ率調整 - 10%～200%の範囲でリサイズ率を指定可能
  • 進捗表示 - プログレスバーで進捗表示
  • 設定記憶 - 前回選択したフォルダを記憶
  • 構造保持 - 元のディレクトリ構造を維持
  • シンプルUI - 直感的で使いやすいインターフェース

🎛️ パラメータ

パラメータ名 範囲 デフォルト値 説明
Source Folder フォルダ選択 - リサイズ対象の画像フォルダを選択します。サブフォルダも含めて処理されます。
Resize Percentage 10% - 200% 50% 画像のリサイズ率を設定します。100%が元のサイズです。
Supported Formats 自動検出 - jpg, jpeg, png, bmp, gif, tiff, webp形式の画像を処理
Output Folder 自動生成 - カレントディレクトリ直下に自動で作成されます
