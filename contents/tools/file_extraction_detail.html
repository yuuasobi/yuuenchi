📁 File Extraction Tool

📖 使い方

File Extraction Toolは、指定した拡張子のファイルのみを抽出し、元のディレクトリ構造を保ったままコピーするツールです。

基本的な使用方法

  1. Pythonで`extract_by_ext.py`を実行（またはexe版を利用）
  2. 「Browse」ボタンで参照元フォルダを選択
  3. 抽出したい拡張子をカンマ区切りで入力（例: txt,png）
  4. ファイル名に含まれる文字列（任意）を入力
  5. 「Extract」ボタンを押すと処理開始
  6. 進捗はプログレスバーで確認
  7. 完了後、保存先のパスが表示されます

⚙️ 機能

  • 複数拡張子対応 - 複数拡張子（例: txt, png など）をカンマ区切りで指定可能
  • ファイル名フィルタ - ファイル名に含まれる文字列でフィルタ（任意）
  • 再帰的検索 - サブフォルダも含めて再帰的にファイルを抽出
  • 構造保持 - 元のディレクトリ構造を維持してコピー
  • 進捗表示 - プログレスバーで進捗表示
  • 設定記憶 - 前回選択した参照フォルダ・拡張子・ファイル名フィルタを記憶・復元
  • 一括管理 - 入力情報は1つの設定ファイルで一括管理
  • 追加保存 - 同じ出力先に複数回抽出しても、前回のファイルは消えず新しいファイルが追加保存

🎛️ パラメータ

パラメータ名 入力形式 必須/任意 説明
Source Folder フォルダ選択 必須 抽出対象のフォルダを選択します。サブフォルダも含めて検索されます。
Extensions カンマ区切り 必須 抽出したい拡張子をカンマ区切りで入力（例: txt,png,jpg）
Filename Filter 文字列 任意 ファイル名に含まれる文字列を指定。空欄の場合は全ファイルが対象
Output Folder 自動生成 - カレントディレクトリ直下に自動で作成されます
