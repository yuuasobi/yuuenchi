🎵 Ghost Tuner

📖 使い方

Ghost Tunerは、リアルタイムで音声のピッチを推定し、モジュレーションエフェクトを適用するプラグインです。

基本的な使用方法

  1. プラグインをDAWに読み込みます
  2. 入力音声を接続します
  3. ピッチ推定の感度を調整します
  4. モジュレーションの強度を設定します
  5. 出力を確認しながらパラメータを微調整します

⚙️ 機能

  • リアルタイムピッチ推定 - 入力音声からリアルタイムでピッチを推定し、高精度な音声処理を実現
  • モジュレーションエフェクト - 推定されたピッチに基づいてモジュレーションエフェクトを適用
  • 低レイテンシー処理 - 最小限の遅延でリアルタイム処理を実現
  • 高品質音声処理 - プロフェッショナルレベルの音質を維持
  • 直感的なUI - 使いやすいインターフェースで簡単に操作可能

🎛️ パラメータ

パラメータ名 範囲 デフォルト値 説明
Pitch Sensitivity 0.0 - 1.0 0.5 ピッチ推定の感度を調整します。高い値ほど細かいピッチ変化を検出します。
Modulation Depth 0.0 - 1.0 0.3 モジュレーションエフェクトの強度を設定します。
Modulation Rate 0.1 - 10.0 Hz 2.0 Hz モジュレーションの速度を調整します。
Input Gain -60.0 - +20.0 dB 0.0 dB 入力信号のゲインを調整します。
Output Gain -60.0 - +20.0 dB 0.0 dB 出力信号のゲインを調整します。
