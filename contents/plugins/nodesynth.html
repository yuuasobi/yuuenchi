🎹 NodeSynth

📖 概要

NodeSynthは、Rustで構築されたノードベースのソフトウェアシンセサイザーVST3/CLAPプラグインです。最大12個の波形ノードを4×3グリッドレイアウトで組み合わせ、個別パラメータコントロールで豊かなサウンドを作成できます。

🎯 使い方

  1. DAWでVST3またはCLAPインストゥルメントとしてプラグインを読み込みます
  2. 波形ボタン（Sine、Square、Triangle、Saw）を使用してノードを追加します
  3. 最大12個までノードを追加できます（制限に達するとボタンが無効化されます）
  4. 各ノードのパラメータを個別に調整します
  5. MIDIノートを演奏してレイヤー合成サウンドを再生します
  6. 各ノードの×ボタンでノードを削除できます

⚙️ 機能

  • ノードベース合成 - 最大12個の波形ノードを組み合わせて豊かなサウンドを作成
  • 4×3グリッドレイアウト - 4列3行の整理された表示で直感的な操作
  • 複数の波形 - サイン、矩形、三角、のこぎり波（色分け表示）
  • 個別ノードパラメータ - 各ノードのゲイン、デチューン、位相、パルス幅
  • MIDI入力 - ノートオン/オフイベントによる完全なポリフォニック対応
  • リアルタイム制御 - 再生中の瞬時パラメータ調整
  • マスターゲイン - 全体出力レベル制御
  • 視覚的フィードバック - 波形名の色分けで簡単識別

🌊 波形タイプ

🔴 サイン波

純粋で滑らかな正弦波音色。最も基本的で自然な音色を提供します。

🟢 矩形波

パルス幅調整可能な豊富な倍音。明るく力強い音色を特徴とします。

🔵 三角波

まろやかで柔らかい倍音成分。温かみのある音色を提供します。

🟡 のこぎり波

明るくバズ感のあるのこぎり波特性。豊かな倍音を含む音色です。

🎛️ パラメータ

パラメータ名 範囲 デフォルト値 説明
Master Gain 0.0 - 1.0 0.7 全体出力レベルを制御します。
Node Gain 0.0 - 1.0 0.5 個別ノードの音量を調整します。
Detune -50 〜 +50 セント 0 セント セント単位のピッチオフセットを設定します。
Phase 0 〜 360度 0度 波形位相オフセットを調整します。
Pulse Width 0 〜 100% 50% 矩形波のデューティサイクルを設定します（矩形波のみ）。

🔧 技術仕様

  • レイアウト - 4×3グリッド
  • ポリフォニック対応 - 完全対応
  • リアルタイム処理 - 低レイテンシー
