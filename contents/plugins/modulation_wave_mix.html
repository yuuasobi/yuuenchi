🎛️ Modulation Wave Mix

📖 使い方

Modulation Wave Mixは、複数の波形を組み合わせて豊かな音色を作り出す波形合成モジュレーションエフェクターです。

基本的な使用方法

  1. プラグインをDAWに読み込みます
  2. 入力音声を接続します
  3. 波形タイプを選択します（サイン波、三角波、矩形波など）
  4. モジュレーションの強度を調整します
  5. 周波数と位相を設定します
  6. 出力を確認しながらパラメータを微調整します

⚙️ 機能

  • 波形合成 - 複数の波形を組み合わせて新しい音色を作成
  • モジュレーションエフェクト - 音声信号に波形を変調して豊かな音色を実現
  • 複数波形対応 - サイン波、三角波、矩形波、ノコギリ波など様々な波形を選択可能
  • リアルタイム処理 - 低レイテンシーでリアルタイム処理を実現
  • 直感的なUI - 波形の視覚的表示と直感的な操作

🎛️ パラメータ

パラメータ名 範囲 デフォルト値 説明
Wave Type Sine, Triangle, Square, Saw Sine モジュレーションに使用する波形の種類を選択します。
Modulation Depth 0.0 - 1.0 0.5 モジュレーションエフェクトの強度を設定します。
Frequency 0.1 - 20.0 Hz 2.0 Hz モジュレーション波形の周波数を調整します。
Phase 0.0 - 360.0° 0.0° モジュレーション波形の位相を調整します。
Mix Level 0.0 - 1.0 0.7 元の信号とモジュレーション信号の混合比率を設定します。
Input Gain -60.0 - +20.0 dB 0.0 dB 入力信号のゲインを調整します。
Output Gain -60.0 - +20.0 dB 0.0 dB 出力信号のゲインを調整します。
