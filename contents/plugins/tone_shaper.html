🎚️ Tone Shaper

📖 概要

Tone Shaperは、XYパッドコントロールでLow/High周波数のトーンシェイピングエフェクトを適用するプラグインです。直感的なXYパッド操作で音色を自由にカスタマイズし、独自のサウンドを作成できます。

🎯 使い方

  1. プラグインをDAWに読み込みます
  2. 入力音声を接続します
  3. XYパッドでLow/High周波数のバランスを調整します
  4. トーンシェイピングの強度を設定します
  5. 周波数範囲を調整します
  6. 出力を確認しながらパラメータを微調整します

⚙️ 機能

  • XYパッドコントロール - 直感的なXYパッドでLow/High周波数のバランスを調整
  • トーンシェイピング - 音色を自由にカスタマイズし、独自のサウンドを作成
  • 周波数範囲制御 - Low/High周波数の範囲を個別に調整可能
  • リアルタイム処理 - 低レイテンシーでリアルタイム処理を実現
  • 視覚的フィードバック - XYパッドの位置と音色の変化を視覚的に確認

🎛️ パラメータ

パラメータ名 範囲 デフォルト値 説明
X Position (Low Freq) 0.0 - 1.0 0.5 X軸でLow周波数の強度を調整します。
Y Position (High Freq) 0.0 - 1.0 0.5 Y軸でHigh周波数の強度を調整します。
Low Frequency Range 20 - 1000 Hz 200 Hz Low周波数の範囲を設定します。
High Frequency Range 1000 - 20000 Hz 8000 Hz High周波数の範囲を設定します。
Shaping Intensity 0.0 - 1.0 0.5 トーンシェイピングエフェクトの強度を設定します。
Q Factor 0.1 - 10.0 1.0 フィルターのQ値（鋭さ）を調整します。
Input Gain -60.0 - +20.0 dB 0.0 dB 入力信号のゲインを調整します。
Output Gain -60.0 - +20.0 dB 0.0 dB 出力信号のゲインを調整します。

🔧 技術仕様

  • 対応フォーマット - VST3、CLAP

  • フィルターエンジン - 高精度デジタルフィルター
  • XYパッド制御 - リアルタイムXY座標処理
